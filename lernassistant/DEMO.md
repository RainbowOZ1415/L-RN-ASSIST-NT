# Demo — 90 Sekunden (Hackathon)

## Ablauf

1. **Sidebar:** Bubble `Klasse 6` wählen
2. **Tab Muster:** Typische Situationen (Chat, Pack-Chance, …)
3. **Fach Deutsch** → Thema **Kommasetzung**
4. **Tab Lehrkraft:** Einstiegsfrage vorlesen — *Jemand schrieb im Chat: 'Er sagte du bist cool.' …*
5. **Fach Mathe** → **Brüche** → Einstiegsfrage
6. **Tab Schüler:** Hook ohne YouTuber-Namen
7. **Login-Toggle** an: „Später personalisierter Feed — heute Bubble-Durchschnitt"
8. **Tab Was läuft?** Kategorie live hinzufügen

## Bubble-Vergleich (30 Sek extra)

- Wechsel zu **Klasse 4–5** → einfachere Fragen
- Wechsel zu **Klasse 10+** → komplexere Fragen

## Pointe

Aus Medienkonsum wird die Neugier-Brücke zum Lehrplan — szenario-basiert, altersgerecht, ohne Creator-Namen.

## Fallback

Sample-Daten laufen offline. Live-Daten: `./run_pipeline.sh` (Keys in `.env`).
